## [1.0.0] - 2026-01-19

### Added

#### Skills
- **autonomous-dev** - Autonomous coding agent that breaks features into user stories
  - PRD generation with clarifying questions
  - JSON conversion for machine-readable tasks
  - Autonomous implementation loop
  - Cross-codebase learning with Memory MCP
  - Worktree integration
  - Smart delegation to specialized agents

- **claude-setup-optimizer** - Analyzes changelog and optimizes your setup
  - Fetches latest Claude Code changelog
  - Analyzes current skills/agents
  - Provides prioritized recommendations
  - Auto-implements approved changes

- **fulltest-skill** - Full-spectrum website testing
  - Site structure mapping
  - Parallel page testing
  - Console error detection
  - Broken link checking
  - Auto-fix capabilities
  - Comprehensive reports

- **worktree-scaffold** - Git worktree automation
  - Feature workspace creation
  - Project scaffolding
  - Parallel development support

#### Agents
- **fulltesting-agent** - E2E testing with Chrome DevTools
- **frontend-agent** - React, Vue, Angular, Next.js specialist
- **api-agent** - REST/GraphQL API specialist
- **database-agent** - Schema, migrations, query optimization
- **devops-agent** - CI/CD, Docker, cloud deployments
- **orchestrator-fullstack** - Multi-layer feature coordination
